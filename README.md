A premium, high-performance AI assistant specialized in **Roblox Scripting** and **Luau** optimization. Built with Nuxt 3 and powered by Groq, this agent provides a sophisticated environment for developers to refine their code, brainstorm features, and manage complex scripting projects.

## ✨ Key Features

- **🧠 specialized Intelligence**: Specifically tuned for Nuxt 3 development and Roblox Luau scripting with a Senior Consultant persona.
- **🚀 Ultra-Fast Responses**: Leverages high-speed LPU inference for near-instant message streaming.
- **📁 Advanced Session Management**:
  - Multi-session chat history.
  - Custom modals for renaming and settings (no native browser prompts).
  - Persistent state via LocalStorage.
- **💎 Premium UI/UX**:
  - **Glassmorphism Design**: Sleek, modern dark-mode aesthetic.
  - **Typewriter Streaming**: Satisfying, organic text-generation animations.
  - **Code specialist**: Rich markdown rendering with syntax highlighting and one-click copy.
- **🔒 Privacy First**: All conversations and API keys are stored securely in your browser's local storage.

## 🛠️ Tech Stack

- **Framework**: [Nuxt 3](https://nuxt.com/)
- **State Management**: [Pinia](https://pinia.vuejs.org/)
- **Styling**: Tailwind CSS & Premium Vanilla CSS
- **AI Engine**: [Groq](https://groq.com/), [Gemini](https://gemini.google.com/app)
- **Icons**: Custom SVG Architecture
- **Markdown**: `markdown-it` with `highlight.js` integration

## 🚀 Getting Started

### Prerequisites

- Node.js (v18.x or later)
- npm, yarn, or pnpm
- A **Groq API Key** (Get it at [Groq Console](https://console.groq.com/keys))
- A **Gemini API Key** (Get it at [Gemini API](https://aistudio.google.com/app/api-keys))

### Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/jipanjiji/LuaScript-AI-Agent.git
   cd LuaScript-AI-Agent
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Configuration**:
   - Navigate to `http://localhost:3000`.
   - Click the **Settings** icon in the sidebar.
   - Enter your **Groq / Gemini API Key** and choose your model (e.g., Llama-3.1-70b / Gemini 3 Pro).
